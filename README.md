# create-catalog

Add [Catalog](https://github.com/interactivethings/catalog) to your app.

## Using yarn

```
yarn create catalog [dir]
```

## Using npx

> `npx` is installed by default with npm >= 5.2.0

```
npx create-catalog [dir]
```

## Using npm

```
npm install -g create-catalog
create-catalog [dir]
```
